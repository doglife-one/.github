# 用語集

DogLife共通の用語定義。

## サービス用語

| 用語 | 説明 |
|------|------|
| DogLife | 愛犬家向けスーパーアプリ |
| DogLife Travel | ペット同伴宿泊予約サービス（OTA） |
| DogLife Column | 愛犬家向けコンテンツメディア |

## UX用語

| 用語 | 説明 |
|------|------|
| DUX | Dog-inclusive User Experience（愛犬中心設計） |
| Journey | カスタマージャーニー（顧客体験の流れ） |
| Blueprint | サービスブループリント（実現構造） |
| BDD | Behavior Driven Development（振る舞い駆動開発） |
| Story | ユーザーストーリー（実装単位） |

## ロール

| 用語 | 説明 |
|------|------|
| user | 旅行者・愛犬家（B2C） |
| admin | DogLife社内（管理者・CS・運用） |
| partner | 施設オーナー・支配人（B2B） |

## 外部サービス
